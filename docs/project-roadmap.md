# Contri Roadmap

## Vision

Build a production-ready expense-sharing platform that helps users track expenses, split bills, and settle balances effortlessly.

---

## Tech Stack

### Frontend
- React
- Tailwind CSS
- React Router
- Axios

### Backend
- Node.js
- Express.js

### Database
- MongoDB Atlas
- Mongoose

### Authentication
- JWT
- HttpOnly Cookies
